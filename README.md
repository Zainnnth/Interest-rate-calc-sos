# Credit Control Interest Calculator

Static GitHub Pages version. No backend is required.

## Deploy

1. Upload these files to the root of the GitHub repository.
2. In GitHub, go to **Settings > Pages**.
3. Source: **Deploy from a branch**.
4. Branch: **main** and folder **/** root.
5. Open the published Pages URL.

## Files

- `index.html` - the application.
- `.nojekyll` - keeps GitHub Pages from applying Jekyll processing.

## Notes

- The calculator runs entirely in the browser.
- The import parser is deliberately limited to prevent the browser freezing on huge ledger exports.
- Interest rates can be edited inside the page using Rate Maintenance.
